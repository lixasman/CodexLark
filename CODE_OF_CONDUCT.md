# Code of Conduct

我们希望 CodexLark 社区保持专业、尊重、务实。

## 我们鼓励的行为

- 基于事实讨论问题
- 尊重不同经验水平的贡献者
- 在提出批评时给出可执行的改进建议
- 关注技术问题本身，而不是针对个人
- 在公开讨论中尽量提供上下文、证据与复现信息

## 不可接受的行为

- 人身攻击、羞辱、骚扰或歧视性言论
- 恶意挑衅、持续性拉扯或故意浪费他人时间
- 公开泄露他人隐私、密钥、日志或敏感环境信息
- 在明知存在安全风险时鼓励不负责任的公开披露
- 以任何形式威胁、恐吓或报复社区成员

## 适用范围

本准则适用于与本项目相关的公开与半公开协作场景，包括但不限于：

- issue
- pull request
- 代码 review
- 讨论区
- 与项目直接相关的演示、答疑和协作文档

## 执行方式

维护者有权删除不当内容、关闭不适当的讨论，或限制反复违反准则的参与者继续互动。

## 如何报告

如果你遇到不当行为，请通过维护者在仓库主页或个人资料中提供的联系方式私下联系，并尽量附上必要上下文。

## 额外说明

技术争论可以尖锐，但表达方式必须克制；标准可以严格，但对人应保持基本尊重。
